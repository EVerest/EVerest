# API module documentation
This module is responsible for providing a simple MQTT based API to EVerest internals

## Connect and disconnect OCPP to the CSMS
This module controls connections to the CSMS over OCPP.

### everest_api/ocpp/cmd/connect
This command starts OCPP connections to the CSMS.

### everest_api/ocpp/cmd/disconnect
This command stops OCPP connections to the CSMS.
