The complete API specification can be found in the

``docs/source/reference/EVerest_API/entrypoint_API.yaml``

file in the source repository, or in the `AsyncAPI HTML documentation <../../../api/entrypoint_API/index.html>`_ automatically generated from it.

Maintenance
===========

The ``apis.yaml`` files knows which API modules exist and some details about them.
The file is not created automatically during the build at this time.
One can use the ``collect_api_detail.py`` to collect information on the existing API modules manually.
